Roofing Hail Intelligence Map

Storm-date hail reports + roof portfolio triage + inspection priority export

MVP v0.1

1. Storm Window

Uses Iowa State IEM archived NWS Local Storm Reports endpoint. These are point reports, not full radar swaths.

2. Roof Portfolio

CSV columns: property, address, lat, lon, roof_type, roof_age, insurer, warranty

3. MESH / Hail Swath Layer

Upload converted MRMS MESH GeoJSON polygons with property hail_inches. This is where true hail circumference/swath goes.
